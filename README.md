## 🎯 Project Overview

OrachdLM is an experimental project aiming to build a Large Language Model (LLM) from scratch, covering the entire pipeline:

- Data collection
- Text preprocessing
- Tokenization
- Model architecture
- Training
- Text generation

The key constraint:
> ⚙️ Everything is designed to run on a single CPU, with no reliance on heavy compute infrastructure.

---

## 🧠 Goals

This project has two main objectives:

### 1. Deep Understanding
Reconstruct every component of a language model to gain a ground-up understanding of how modern LLMs work.

### 2. Efficiency & Optimization
Explore novel optimizations across the pipeline under strict computational constraints:
- Memory efficiency
- Training speed
- Lightweight architectures
- Minimal dependencies

-  Text generation interface  

---

## ⚙️ Design Principles

- Simplicity first — minimal abstractions, maximum clarity  
- From scratch — avoid black-box frameworks when possible  
- Efficiency-aware — every step considers CPU constraints  
- Modular — each component can be studied independently
